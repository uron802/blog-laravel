## Install Laravel
* composer update
* cp .env.example .env
* php artisan key:generate
* .envファイルのデータベースの設定を各々の設定に変更
* php artisan migrate
* npm install
* npm run dev

## Install Laravel File Manager
* php artisan storage:link

## Install
* cd hogehoge
* git clone https://github.com/uron802/blog-laravel.git
* cd blog-laravel/src
* composer update
* cp .env.example .env
* php artisan key:generate
* .envファイルのデータベースの設定を各々の設定に変更
* php artisan migrate
